# 🎲 Dice Roller Program

This repository contains a simple Python program that simulates rolling two dice.  
It’s a beginner-friendly project showcasing **loops, conditionals, user input handling, and random number generation**.

---

## ✨ Features
- Prompts the user to roll the dice (`y/n`).
- If **yes**, generates two random numbers between 1 and 6 and prints them.
- If **no**, thanks the user and terminates the program.
- Handles invalid choices gracefully with an error message.
